# Shikhar Dixit — Portfolio

Personal portfolio website built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies, no backend.

## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository (e.g., `shikhar-portfolio`)
2. Upload all files:
   - `index.html`
   - `photo.jpeg`
   - `README.md`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)`
5. Click **Save** — your site will be live at `https://<your-username>.github.io/<repo-name>`

## 📁 Files
- `index.html` — complete portfolio (self-contained)
- `photo.jpeg` — profile photo
- `README.md` — this file

## ✨ Features
- Responsive design (mobile + desktop)
- Smooth scroll-reveal animations
- Active nav highlight on scroll
- Dark theme with accent glow effects
- All sections: Hero, About, Skills, Projects, Education, Contact
